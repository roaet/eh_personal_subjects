## Mac Setup

### Steps

0. chsh /bin/bash
1. git clone dot files
2. install brew
3. install pyenv and install python versions
4. update .bash_extended with pyenv  # fix this in dot files plz
5. install go
6. install spacevim
7. install tmux
8. install eh
